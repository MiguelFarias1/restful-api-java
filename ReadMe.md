# Projeto Spring Boot

Este é um projeto Spring Boot criado para estudos em SpringBoot com Java, com ênfase em: JPA, padrão REST, Hateoas, Migrations, padrão DTO, CORS, testes com JUnit, JWT e Spring Security.

## Tecnologias Utilizadas

- **Java 21**
- **Spring Boot**
- **MySQL**
- **Spring Data JPA**
- **Lombok**
- **Swagger**
- **JUnit**
- **Mockito**
- **JWT**
- **Spring Security**
